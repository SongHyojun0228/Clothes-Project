## 📌 프로젝트 소개
### <ins>MYCLOSET는 패션을 좋아하는 사람들을 위한 커뮤니티 플랫폼입니다.</ins>
- MYCLOSET 사용자들은 발매 정보나 착장 소개 등과 같은 패션 정보를 얻을 수 있습니다.
- 또한, 다양한 의견, 가치관을 가진 사람들끼리 소통할 수 있습니다.</p>

### API 명세서

<br>
<br>

<h2>📌 선정이유</h2>
<p>평소에 옷에 관심이 많아서 패션 커뮤니티를 만들어보고 싶다는 생각으로 시작하게 된 개인 프로젝트입니다.</p>
<p>다양한 패션 정보와 스타일을 공유하고 의견을 나눌 수 있는 공간을 구현해보고 싶었고, 그 과정에서 Node.js와 Express를 이용해 게시판 기능을 직접 설계하고 구현하면서 웹 개발의 기본 흐름을 익히는 데 큰 도움이 되었습니다.</p>

<br><br>

<h2>📌 주요기능</h2>
<h3>🙋🏻‍♂️ 회원기능</h3>
<p>- 회원가입 시 DAUM 주소 API 활용</p>
<p>- 미이페이지에서 프로필 사진 변경</p>
<p>- 마이페이지에서 로그인한 유저가 작성한 글 리스트</p>

<h3>💬 커뮤니티</h3>
<p>- 옷과 관련된 오뭐신, 오늘의 룩, 골라줘 커뮤니티</p>
<p>- 라이프 스타일의 자유, 밥밥, 운동, 유머, 일상 커뮤니티</p>
<p>- 오늘 올라온 글은 'n' 배찌로 구분</p>
<p>- AJAX를 이용한 댓글 등록</p>
<p>- AJAX를 이용한 좋아요 등록</p>

<br>
<br>

<h2>📌 성장점</h2>
<p>AJAX를 활용한 댓글, 좋아요 기능 구현을 통해 새로고침 없이 동작하는 웹페이지를 만들 수 있었습니다./p>
<p>커뮤니티 분리, 작성일에 따른 배찌 표시 등의 기능을 구현하면서 UI 구성에 대해 고민할 수 있었고, 실제로 사용해보면서 어떤 기능이 필요한지 직접 느끼고 반영해보는 경험을 해봤습니다.</p>

<h2>🛠️ 설치 및 실행 방법</h2>
<h3>1️⃣ 프로젝트 클론</h3>

```
git clone https://github.com/SongHyojun0228/Clothes-Project.git
cd Clothes-Project
```

<h3>2️⃣ 패키지 설치</h3>

```
npm install
```

<h3>3️⃣ 서버 실행</h3>

```
nodemon app.js
```

<h3>4️⃣ 로컬 환경에서 접속</h3>

```
http://localhost:3000
```
