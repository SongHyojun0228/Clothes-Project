<h1>👖 MYCLOSET - 패션 커뮤니티</h1>

<br>
<br>

<h2>📌 프로젝트 소개</h2>
<h3><ins>MYCLOSET는 패션을 좋아하는 사람들을 위한 커뮤니티 플랫폼입니다.</ins></h3>
<p>MYCLOSET 사용자들은 발매 정보나 착장 소개 등과 같은 패션 정보를 얻을 수 있습니다.</p>
<p>또한, 다양한 의견, 가치관을 가진 사람들끼리 소통할 수 있습니다.</p>

<br>
<br>

<h2>주요기능</h2>
<h3>💬 커뮤니티</h3>
<p>- 옷과 관련된 오뭐신, 오늘의 룩, 골라줘 커뮤니티</p>
<p>- 라이프 스타일의 자유, 밥밥, 운동, 유머, 일상 커뮤니티</p>

<h2>🛠️ 설치 및 실행 방법</h2>
<h3>1️⃣ 프로젝트 클론</h3>

```
git clone https://github.com/SongHyojun0228/Clothes-Project.git
cd Clothes-Project
```

<h3>2️⃣ 패키지 설치</h3>

```
npm install
```

<h3>3️⃣ 서버 실행</h3>

```
nodemon app.js
```

<h3>4️⃣ 로컬 환경에서 접속</h3>

```
http://localhost:3000
```
